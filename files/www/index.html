What is MEOW TOWN?

MEOW TOWN is a computer that is shared amongst some friends!

You can use it to host a webpage, chat with others, or just mess around.

Why is MEOW TOWN?

Inspired by tilde.club, MEOW TOWN seeks to build a community that is:

  • friendly
  • creative
  • DIY-oriented

Who is MEOW TOWN?

The current list of active members is:

  • max

What are the rules?

All members are expected to read and follow the code of conduct. Otherwise, please be kind and considerate of others and the very limited resources of the system.

How can I join MEOW TOWN?

Send an email to mxhold+meowtown@gmail.com with the subject "New user: yourusername". Please pick a username that is fewer than 8 characters.

In the body of the email, please say "I have read and agree to follow the code of conduct."

MEOW TOWN is just starting up so it might take a while for us to add you.

How was MEOW TOWN set up?

MEOW TOWN is running FreeBSD 10.3. See the setup page for more details.
